## Exercício de Fixação - Transmitindo informações entre componentes

### **Proposta**

- No formulário, criado nos exercícios anteriores, fazer com que dois campos sejam componentes filhos do componente pai do formulário. Garantir que o estado ainda pertença ao componente pai.

- Fazer validações nos componentes criados.

- Criar no estado do componente pai, um campo formularioComErros o qual deve ser true, caso algum desses componentes tenha erros, e false caso os componentes não tenham erros.
